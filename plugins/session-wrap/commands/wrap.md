# Session Wrap

セッション終了時に作業内容を要約して保存する。

## Instructions

1. **ディレクトリ確認**: `.claude/sessions/` ディレクトリが存在しない場合は作成する

2. **会話の要約**: このセッションでの作業内容を振り返り、以下の4項目を整理する:
   - **やったこと**: 完了したタスク、実装した機能、修正したバグなど
   - **途中のこと**: 作業中だが完了していないタスク
   - **次やること**: 次のセッションで取り組むべきタスク
   - **メモ**: 注意点、発見したこと、覚えておくべきこと

3. **ファイルに追記**: `.claude/sessions/YYYY-MM-DD.md` に以下の形式で追記する:

```markdown
## YYYY-MM-DD HH:MM

### やったこと
- [完了したタスクをリスト]

### 途中のこと
- [作業中のタスクをリスト]

### 次やること
- [次に取り組むタスクをリスト]

### メモ
- [覚えておくべきことをリスト]
```

4. **確認メッセージ**: 保存完了後、要約内容をユーザーに表示する

## Notes

- 日付は現在の日付を使用する
- 時刻は24時間形式で記録する
- 同じ日に複数回実行した場合は、時刻付きで追記する
- 内容がない項目は「特になし」と記載する
- 日本語で記述する
