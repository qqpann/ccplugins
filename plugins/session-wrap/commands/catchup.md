# Session Catchup

セッション開始時に前回の作業状況を確認する。

## Instructions

1. **ログファイルの検索**: `.claude/sessions/` ディレクトリ内の最新の `.md` ファイルを探す
   - Glob パターン: `.claude/sessions/*.md`
   - ファイルがない場合は「ログがありません。新しいセッションを始めましょう！」と表示して終了

2. **ログの読み取り**: 最新のログファイルを読み取る

3. **状況の要約**: ログ内容を分析し、以下を簡潔に伝える:
   - 前回いつ作業したか
   - 何をやっていたか（途中のこと）
   - 次に何をする予定だったか
   - 特記事項（メモ）

4. **セッション開始の促し**: 要約後、以下のように問いかける:
   ```
   ---
   何から始めますか？
   ```

## Output Format

```
📋 前回のセッション: YYYY-MM-DD HH:MM

**途中だったこと:**
- [作業中だったタスク]

**次やる予定だったこと:**
- [予定していたタスク]

**メモ:**
- [注意点など]

---
何から始めますか？
```

## Notes

- ログがない場合は新規セッションとして扱う
- 複数エントリがある場合は最新のものを参照する
- 簡潔に要約し、詳細はユーザーが求めた場合に提供する
